# duet

Wireless unibody portable keyboard.

## Features

- wireless (nice!nano MCU)
- unibody that can be disassembled into two parts for traveling
- 40 Choc V1 switches
- 1 rotary encoder and 1 five-way switch
- 3D printed case
- nice!view support
